# Športový deň 2026 | Tribe Home Experience

Plná verzia jednoduchej eventovej mini appky postavenej na Next.js 14.

## Lokálne spustenie

```bash
npm install
npm run dev
```

## Deploy na Vercel

1. Nahraj tento obsah do nového GitHub repozitára.
2. Vo Verceli zvoľ **New Project**.
3. Vyber repo a potvrď deploy.
4. Framework sa rozpozná automaticky ako Next.js.

## Kde meniť obsah

Väčšina obsahu je v `app/page.tsx`.
